# COAST-XCHANGE Viz 🌊🧭

A tiny but mighty home for the **COAST-XCHANGE secondment skills visualisation**.

This repository hosts:
- the interactive web guide,
- the institution-based skills explorer,
- and the coding logic used to describe secondments across the consortium.

## Live site
👉 https://defreitasl.github.io/COAST_XCHANGE_viz/

## What is this?
A lightweight visual tool to help consortium members:
- browse skills by institution,
- identify relevant secondment codes,
- and describe secondments in a more consistent way.

## Status
Still evolving, but already useful.
